# GigaChat Clone

Чат-приложение на React с интеграцией OpenRouter API, вдохновлённое ChatGPT.

## Функциональность

- ✅ Отправка сообщений к нейросети (OpenRouter)
- ✅ Потоковый вывод ответов (streaming)
- ✅ Markdown-форматирование ответов (заголовки, списки, код)
- ✅ Подсветка синтаксиса в блоках кода
- ✅ Индикатор печати (Typing Indicator)
- ✅ Автоматическая прокрутка к последнему сообщению
- ✅ Кнопка копирования ответов ассистента
- ✅ Боковая панель с управлением чатами
- ✅ Создание, переключение и удаление чатов
- ✅ Автоматическое сохранение в localStorage

## Технологии

- React 18 + TypeScript
- Vite
- CSS Modules
- OpenRouter API
- react-markdown + remark-gfm + rehype-highlight
